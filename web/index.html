🌼 闲聊花 Web

一朵话很多的小花:自动闲聊 · 整点报时 · 计划提醒

戳一下小花,它会回应你

浏览器需要你点一下,才允许网页发出声音

设置

在 elevenlabs.io 免费注册取 API Key,再从 Voice Library 选一个可爱音色复制它的 Voice ID。Key 只保存在你本地浏览器,不会上传别处。

预设

小花会隔一会儿自己说点什么

每到整点,小花会报一次时间

睡觉时间内不闲聊、不报时(计划提醒仍然生效)

自定义文案

添加你自己的闲聊台词,会和内置台词一起被随机说出来(按当前语言分别管理)。

关掉后,小花只说你写的台词

    还没有自定义台词,添加一条试试吧

    计划提醒

    到点时小花会大声念出来,并弹出通知(需要保持本页面打开;后台常驻请用浏览器插件版)

      还没有计划,添加一条试试吧

      粉丝自制网页玩具,与任天堂无关;台词均为原创,语音来自浏览器自带 TTS 或你自选的在线引擎。
